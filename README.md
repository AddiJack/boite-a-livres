# 📚 La Boîte à Livres — Outil de recherche de livres

Un outil de recherche intégrable dans Notion pour trouver facilement les infos d'un livre (titre, auteur, nombre de pages, genre) et les copier en un clic dans ta bibliothèque.

## ✨ Fonctionnalités
- Recherche par titre, auteur ou ISBN
- Données via Google Books API (millions de titres)
- Copie en un clic : tout ou champ par champ
- Détection automatique du genre
- Intégrable dans Notion via un bloc `/embed`

## 🚀 Utilisation
1. Déploie via GitHub Pages
2. Copie l'URL générée (`https://ton-pseudo.github.io/boite-a-livres/`)
3. Dans Notion, tape `/embed` et colle l'URL

## 🛠️ Stack
HTML · CSS · JavaScript vanilla · Google Books API
