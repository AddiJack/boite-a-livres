📖 Comment trouver tes infos Notion

1. Ton Token Notion :

  1. Va sur notion.so/my-integrations
  2. Clique sur ton intégration "Boîte à Livres API"
  3. Copie le "Internal Integration Secret"
ntn_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx ou secret_xxxxxxxxxx...

2. Ton Database ID :

  1. Dans Notion, trouve "Ma Bibliothèque Personnelle" dans le menu gauche
  2. Fais un clic droit dessus → "Copier le lien"
  3. L'URL ressemble à ça :
notion.so/workspace/Ma-Bibliotheque-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx?v=...

👆 La partie surlignée c'est ton Database ID — les 32 caractères juste avant le ?v=.

✨ Astuce : tu peux aussi coller l'URL complète ici, l'outil extrait l'ID tout seul !
